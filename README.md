# ERP Coffee V1

Учебный мини-проект ERP для кофейни на `React + Vite`.

## Что делает проект

- показывает меню напитков (например, латте, капучино, американо);
- хранит список ингредиентов и остатки на складе;
- для каждого напитка использует рецепт (какие ингредиенты и сколько нужно);
- при нажатии на кнопку приготовления:
  - сохраняет заказ;
  - списывает ингредиенты со склада;
  - пересчитывает, сколько порций каждого напитка еще можно приготовить;
- показывает предупреждения, когда ингредиенты подходят к минимальному порогу;
- сохраняет состояние в `localStorage` (после перезагрузки данные не теряются).

## Технологии

- `React`
- `Vite`
- без дополнительных UI-библиотек и сложных паттернов

## Запуск проекта

1. Установите зависимости:

```bash
npm install
```

2. Запустите dev-сервер:

```bash
npm run dev
```

3. Откройте адрес из консоли (обычно):

`http://localhost:5173`

## Проверка работы V1

- Нажмите `Приготовить` у любого доступного напитка.
- Проверьте, что:
  - заказ появился в блоке «Последние заказы»;
  - остатки ингредиентов уменьшились;
  - показатель «Можно приготовить» обновился;
  - при низких остатках появился блок предупреждений.

## Production сборка

```bash
npm run build
```
